** c splash splash - map style editor **

Developed to study CSS/SLD languages and create a render environment client side for map styles using sld body and canvas 2D

** install modules **

`npm install`

** development **

`npm start`

url -> **http://localhost:3000/**

** build **

`npm run compile`

output -> **dist/cspsp.js**
